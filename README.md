# Portfolio

The idea behind this project was to move away from the traditional single-page portfolio with endless scrolling. Instead, I wanted to explore a different approach where navigation itself becomes part of the experience.

Inspired by desktop environments, this portfolio uses draggable windows and interactive elements to encourage exploration and present my work in a way that feels more natural to me.

## Features

* Desktop-inspired interface;
* Draggable, minimizable, and resizable windows;
* Dedicated experience for mobile devices;
* Portuguese and English language support;
* Light and dark themes;
* Subtle sound effects for selected interactions;
* Project presentations focused not only on the final result, but also on the process and decisions behind each solution.

## Tech Stack

### Frontend

* React 19
* Vite 7
* Tailwind CSS 4
* Framer Motion

### State Management

* Zustand

### Styling

* Tailwind CSS
* CSS Variables for theming
* Responsive layouts with dedicated desktop and mobile implementations

### Internationalization

* Custom translation system with support for English and Portuguese

### Deployment

* GitHub Pages
* GitHub Actions for automated deployment

## Notes

This project will probably continue to evolve over time as I learn new things, revisit old decisions, and refine the experience. It represents where I am today as a developer and designer, and serves as a place to share not only what I build, but also how I approach solving problems.
